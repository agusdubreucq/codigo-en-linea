la etiqueta <b> pone en negrita una palabra sin darle importancia semantica

la etiqueta <strong> tambien pone en negrita la palabra, pero le da mas importancia semantica

la etiqueta <i> pone el texto en italica

la etiqueta <em> es para dar enfasis y se puede anidar al texto

el <br> es para un salto de linea como
aca, pero no usar para incrementar espacios entre lineas

los <li> declara cada elemento de una lista

  • primero
  • segundo

    • los <ol> definen listas ordenadas

      primero
      segundo

    los <ul> crea lista no ordenada

    • se usa con el <li>

    el <div> es para contener otros elementos html

    el <img> agrega una imagen y tiene los atributos <img src="imagen.png alt="descripcion de imagen>

    el <a> es para enlaces a otras paginas o ubicaciones dentro de la misma pagina home eso lo hice con <a href="/">
